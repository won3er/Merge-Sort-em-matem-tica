# Equação de 2º Grau

**Descrição**:  
Uma equação de 2º grau tem a forma ax² + bx + c = 0, onde a, b e c são constantes.

**Exemplo**:  
Resolver x² - 5x + 6 = 0:  
- Fatorando: (x - 2)(x - 3) = 0, logo x = 2 ou x = 3.

**Aplicações**:  
- Engenharia: cálculo de trajetórias.  
- Economia: análise de máximos e mínimos.
