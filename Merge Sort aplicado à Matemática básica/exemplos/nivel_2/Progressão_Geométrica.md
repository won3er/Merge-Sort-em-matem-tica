# Progressão Geométrica (PG)

**Descrição**:  
É uma sequência de números em que o quociente entre termos consecutivos é constante.

**Exemplo**:  
PG: 3, 6, 12, 24 (razão = 2).

**Aplicações**:  
- Economia: juros compostos.  
- Física: crescimento exponencial.
