# Progressão Aritmética (PA)

**Descrição**:  
É uma sequência de números em que a diferença entre termos consecutivos é constante.

**Exemplo**:  
PA: 2, 4, 6, 8 (razão = 2).

**Aplicações**:  
- Economia: análise de rendimentos constantes.  
- Engenharia: cálculos estruturais.
