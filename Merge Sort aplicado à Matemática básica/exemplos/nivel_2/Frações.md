# Frações

**Descrição**:  
Uma fração representa uma parte do todo e é expressa como a/b, onde 'a' é o numerador e 'b' é o denominador.

**Exemplo**:  
Soma de frações:  
1/2 + 1/3 = 3/6 + 2/6 = 5/6.

**Aplicações**:  
- Cálculos de proporções.  
- Divisão de recursos de forma justa.
