# Análise Combinatória

**Descrição**:  
Estuda as diferentes formas de agrupar ou ordenar elementos.

**Exemplo**:  
Quantos modos de organizar 3 livros em uma prateleira?  
3! = 3 × 2 × 1 = 6.

**Aplicações**:  
- Probabilidade: cálculo de eventos.  
- Computação: algoritmos de permutação.
