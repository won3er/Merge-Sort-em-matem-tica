# Logaritmos

**Descrição**:  
Um logaritmo é o expoente ao qual a base deve ser elevada para produzir um número.

**Exemplo**:  
log₂(8) = 3, porque 2³ = 8.

**Aplicações**:  
- Matemática financeira: cálculo de juros compostos.  
- Computação: algoritmos eficientes.
